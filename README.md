# Finance Dashboard Backend

## 📌 Overview
This project is a backend API for managing financial transactions. It supports basic CRUD operations.

## 🚀 Tech Stack
- Node.js
- Express.js

## 📂 Features
- Add transaction
- Get all transactions

## ▶️ How to Run
1. Install dependencies:
   npm install

2. Start server:
   node server.js

## 🔗 API Endpoints
- POST /transactions
- GET /transactions

## 📸 Testing
All APIs are tested using Postman and working successfully.
